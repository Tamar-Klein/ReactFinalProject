# 🎧 HelpDesk Support System

מערכת ניהול פניות (Ticketing System) מקצועית ומודרנית, המאפשרת תקשורת יעילה בין לקוחות, נציגי שירות ומנהלים לפתרון תקלות וניהול בקשות.

---

## 🚀 מה המערכת עושה?

* **ניהול מחזור חיי פנייה** – יצירה, מעקב, עדכון וסגירת פניות שירות.
* **תעדוף וסינון חכם** – חיפוש חופשי, סינון לפי סטטוס ומיון לפי רמת דחיפות.
* **ממשק משתמש מתקדם** – עיצוב נקי ומודרני המבוסס על Material UI, כולל מצבי טעינה וטיפול בשגיאות.
* **תקשורת פנימית** – מערכת תגובות לכל פנייה לצורך מעקב ועדכון שוטף.

---

## 👥 תפקידים במערכת

### 👤 לקוח (Customer)

* פתיחת פניות חדשות
* בחירת רמת דחיפות
* מעקב אחר סטטוס הפניות האישיות

### 🧑‍💼 נציג (Agent)

* צפייה בפניות שהוקצו אליו
* עדכון הטיפול בפנייה
* שינוי סטטוסים (למשל: "בטיפול", "סגור")

### 🛡 מנהל (Admin)

* שליטה מלאה במערכת
* ניהול משתמשים
* הגדרת רמות דחיפות וסטטוסים גלובליים
* צפייה בכלל הפניות בארגון

---

## 🛠 הנחיות הרצה

### 1️⃣ צד השרת (Backend)

**מאגר:** `helpdesk-api`

שכפול המאגר:

```bash
git clone https://github.com/sarataber/helpdesk-api.git
cd helpdesk-api
```

התקנת תלויות:

```bash
npm install
```

הגדרת קובץ `.env` עם:

* פרטי מסד הנתונים
* `JWT_SECRET`

הרצת השרת:

```bash
npm start
```

---

### 2️⃣ צד הלקוח (Frontend)

**מאגר:** `FinalProject-React`

שכפול המאגר:

```bash
git clone https://github.com/Tamar-Klein/FinalProject-React.git
cd FinalProject-React
```

התקנת תלויות:

```bash
npm install
```

הרצת האפליקציה:

```bash
npm run dev
# או
npm start
```

האפליקציה תהיה זמינה בדפדפן בכתובת:

```
http://localhost:5173
```

או:

```
http://localhost:3000
```
